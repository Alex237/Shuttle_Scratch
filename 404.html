Qu'est ce que Shuttle ?

Shuttle est une solution web libre de gestion de Support. Il est destiné à des services de HelpDesk, multi-techniciens ou pas. Cette outil permet de centraliser, classifier, vos demandes et incidents par l'intermédiaire de tickets, vous permettant ainsi d'informer en temps réel vos collaborateurs.
Au fur et à mesure de la création de vos incidents vous mettez en place une base de connaissance, qui vous permettra de trouver rapidement des réponses à vos demandes.

Prérequis

Système d'exploitation :

Linux x86 (32 bits) and x86-64 (64 bits) with Apache 2.2 and Nginx web servers ; Windows x86 (32 bits) and x86-64 (64 bits), Windows 7, Windows 8, Windows Server 2008 R2, Windows XP SP3 ; Mac OS X

Les serveurs Web tournant avec :

Apache 2.x (Sous Windows, Mac et Linux) ; Nginx web server ; Microsoft IIS (version windows le supportant) ; Serveurs MySQL 5.5 or 5.6 server ; PHP

Les navigateurs suivant :

Google Chrome (recommandé) ; Mozilla Firefox (recommandé) ; Microsoft Internet Explorer 9 and 10 ; Apple Safari (recommandé)
